ACL (Access Control List)

A list of permissions attached to a resource that specifies which users or groups can access it and what actions they can perform.

Active Directory (AD)

Microsoft's directory service for managing users, computers, and policies in a Windows domain network.

AES (Advanced Encryption Standard)

A symmetric encryption algorithm widely used to protect data. Supports 128, 192, and 256-bit key sizes.

ALE (Annualized Loss Expectancy)

The expected monetary loss from a specific risk over one year. Calculated as SLE × ARO.

APT (Advanced Persistent Threat)

A sophisticated, long-term cyberattack campaign, typically conducted by nation-state or well-funded actors.

ARO (Annualized Rate of Occurrence)

The estimated number of times a specific threat is expected to occur within a one-year period.

Attack Vector

The method or pathway used by an attacker to gain unauthorized access to a system.

Bash

Bourne Again Shell — the default command-line interpreter on most Linux systems, widely used for scripting.

BitLocker

Microsoft's full-disk encryption feature built into Windows Professional and Enterprise editions.

Brute Force Attack

An attack method that systematically tries all possible passwords or keys until the correct one is found.

C2 (Command and Control)

The infrastructure and communication channels attackers use to maintain control over compromised systems.

CIA Triad

The three pillars of information security: Confidentiality, Integrity, and Availability.

CIS Controls

A prioritized set of actions developed by the Center for Internet Security that form a defense-in-depth set of best practices for cybersecurity.

CompCert

CompTIA Competency Certificate — a credential earned by completing a structured course and assessments.

Credential Stuffing

An attack that uses stolen username/password pairs from previous breaches to attempt login on other services.

Cron

A Linux utility that schedules scripts and commands to run automatically at specified intervals.

CVSS

Common Vulnerability Scoring System — a standardized severity rating for security vulnerabilities (0.0–10.0).

DDoS

Distributed Denial of Service — an attack that overwhelms a target with traffic from multiple sources.

Defense in Depth

A security strategy employing multiple layers of controls so that failure of one layer does not compromise the whole system.

DHCP

Dynamic Host Configuration Protocol — automatically assigns IP addresses to devices on a network.

DLP (Data Loss Prevention)

Technologies and policies that prevent sensitive data from leaving an organization's control.

DNS

Domain Name System — translates human-readable domain names into IP addresses.

EDR (Endpoint Detection and Response)

Advanced endpoint security that provides continuous monitoring, threat detection, and automated response.

Encryption

The process of converting plaintext into ciphertext using an algorithm and key, making it unreadable without the correct key.

EPP (Endpoint Protection Platform)

A security solution that protects endpoints (workstations, servers, mobile devices) from threats using antivirus, anti-malware, and firewall capabilities.

Firewall

A network security device that monitors and filters traffic based on a defined set of security rules.

GPG (GNU Privacy Guard)

An open-source implementation of the OpenPGP standard for encrypting and signing data and communications.

GPO (Group Policy Object)

A collection of security and configuration settings applied to users and computers in an Active Directory domain.

Hashing

A one-way function that produces a fixed-size output (digest) from any input. Used for integrity verification and password storage.

HTTPS

HTTP Secure — web browsing encrypted with TLS, providing confidentiality, authentication, and integrity.

IDS (Intrusion Detection System)

A system that passively monitors network traffic or host activity and alerts on suspicious patterns.

IoC (Indicator of Compromise)

Forensic evidence that suggests a system or network has been breached (e.g., malicious IPs, file hashes).

IPS (Intrusion Prevention System)

A system that monitors traffic inline and can actively block detected threats.

ISO 27001

An international standard for information security management systems (ISMS), specifying requirements for establishing, implementing, and continually improving security.

Lateral Movement

The technique attackers use to move through a network after initial compromise, accessing additional systems.

MITRE ATT&CK

A knowledge base of adversary tactics and techniques based on real-world observations, used for threat analysis.

MFA (Multi-Factor Authentication)

An authentication method requiring two or more verification factors (something you know, have, or are).

NAT (Network Address Translation)

Translates private IP addresses to public IP addresses, allowing multiple devices to share a single public IP.

NIST CSF

National Institute of Standards and Technology Cybersecurity Framework — five-function model: Identify, Protect, Detect, Respond, Recover.

Nmap

Network Mapper — open-source tool for network discovery, port scanning, and security auditing.

Phishing

A social engineering attack using deceptive emails or websites to trick users into revealing sensitive information.

Ransomware

Malware that encrypts a victim's files and demands payment (ransom) for the decryption key.

RPO (Recovery Point Objective)

The maximum acceptable amount of data loss measured in time. Determines backup frequency.

RTO (Recovery Time Objective)

The maximum acceptable time to restore service after a disruption.

SIEM

Security Information and Event Management — collects, correlates, and analyzes logs from multiple sources for threat detection.

SLE (Single Loss Expectancy)

The expected monetary loss each time a risk event occurs. Calculated as Asset Value × Exposure Factor.

SOC (Security Operations Center)

A centralized team that monitors, detects, and responds to cybersecurity threats 24/7.

SQL Injection

An attack that inserts malicious SQL code through application input fields to manipulate databases.

SSH (Secure Shell)

An encrypted protocol for secure remote access to systems (port 22).

SUID

Set User ID — a Linux permission bit that allows a program to run with the file owner's privileges.

Supply Chain Attack

An attack that compromises trusted third-party software, hardware, or service providers to gain access to their customers' environments.

TLS (Transport Layer Security)

A cryptographic protocol that provides secure communication over networks. The successor to SSL.

UAC (User Account Control)

A Windows security feature that prompts for administrator confirmation before allowing system changes.

VLAN

Virtual Local Area Network — a logical network segment on a managed switch for traffic separation.

VPN (Virtual Private Network)

Creates an encrypted tunnel between a client and server for secure communication over untrusted networks.

Vulnerability

A weakness in a system, application, or process that could be exploited by a threat actor.

Wireshark

An open-source network protocol analyzer for capturing and inspecting network traffic.

WPA3

Wi-Fi Protected Access 3 — the current standard for wireless network encryption and authentication.

Zero-Day

A vulnerability that is unknown to the vendor and has no available patch, making it particularly dangerous.
